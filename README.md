# Excel USA Car Sales Analysis

Excel-based data analysis project focused on cars sold in the United States in 2019.

## Project background
The goal of this project was to analyze automotive sales data and answer business and product-related questions about the vehicles offered on the US market in 2019.

The dataset contains detailed information about cars, including manufacturer, model year, category, price, fuel consumption, engine characteristics, transmission, and selected technical features.

## Analysis focus
The analysis is focused on questions such as:
- number of cars offered in 2019
- age structure of offered cars
- manufacturer-level vehicle counts
- engine size and model year relationships
- fuel consumption and engine displacement
- car price and horsepower
- relationships between vehicle weight, dimensions, and other technical characteristics
